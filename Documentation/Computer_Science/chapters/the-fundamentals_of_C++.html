The Fundamentals of C++

The following is the exposition to C++. To write your own code you will need a text editor, which like the name states, is an application that helps you edit text. Luckily, all OS (Operating Systems) come with one! For example, Windows has Notepad, MacOS has TextEdit, and Linux usually has Vim. Now I say "usually" since there are multiple distributors that preload the Linux OS with there own choice of editor, but usually you will find Vim, especially if you use Ubuntu. There are also third party editors that come with extra features. The one I am currently using is Microsoft's Visual Studio Code.
